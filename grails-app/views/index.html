Bucket List

Project description

This project is a bucket list, where a user can individually manage his goals.

The features are as follows:

  • Goals consisting of:
    • Description (text)
    • Image (only .jpg, with "upload")
    • Deadline (date)
    • Priority (out of 1, 2 or 3)
    • State (reached / not yet reached)
    • Topic (out of the existing topics)
  • Methods for goals:
    • Add goals
    • Edit goals
    • Delete goals
  • Topics consisting of:
    • Title (text)
  • Methods for topics:
    • Add goal topics
    • Edit goal topics
    • Delete goal topics
